What is the TGEngine?

It's planned to become a game engine. The Github repository can be found here.

Status

Our roadmap can be found here. Everything is still being worked on. Please follow the official Troblecodings Twitter account for updates.

Build on your own

The project dependencies are required to be installed in order to compile your own build. That can be done by running python setup.py. Select 1 Get dependencies to install the required dependencies. Then you should be able to open the Visual Studio project file and compile the Engine.

Libraries

I am using LunarG's Vulkan-SDK.
Current build version: 1.1.130
Current Vulkan API version: 1.0

STB is currently used for image and font loading. This is going to be replaced in the future.

SIDE NOTE: The file library TGVL may be needed in the future to compile newer versions of the TGEngine, but it is not supported at the current date!
