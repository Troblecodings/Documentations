Documentation of the troble game resource (tgr) files

Version 1

The file starts with a 32 bit version number as header. Following that are the content blocks

Every block starts with a 32bit length of the following content after that follows the content

If it should follow a 0xFFFFFFFF after a block it will continue to the next content type. Content type order will be the following

  • Textures
  • Materials
  • Actors
  • Animations
